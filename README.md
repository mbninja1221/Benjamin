## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- TypeScript
- Three.js (for 3D elements)
- React Type Animation
